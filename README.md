# ComfyCafe

A Cafe Shop web application that integrates online ordering and payment systems for customer transactions.

**Features**
----
- Product Display
- Stripe Payments
- Mobile Responsive

**Tech Stack**
----
Frontend: ReactJS, Tailwind, Typescript

Backend: Express JS, PostgreSQL

Hosting Services: Vercel Client-side, Render Server-side, RenderDB
