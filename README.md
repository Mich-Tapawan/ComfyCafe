# ComfyCafe

A Cafe Shop web application that integrates online ordering and payment systems for customer transactions.

**Features**
----
- Product Display
- Admin Panel / Control
- Stripe Payments
- Token-based Authentication and Authorization
- Mobile Responsive

**Tech Stack**
----
Frontend: ReactJS, Tailwind, Typescript

Backend: Express JS, MQL (MongoDB)

Hosting Services: Vercel Client-side, Render Server-side, MongoDB
